# Aloo - the base to your *torkari* (project).

An easy starting point with ReactJS, SASS and Redux.

## Installation

**Prerequisites**
- Yarn

```bash
  $ git clone https://github.com/faisalhmohd/aloo.git
  $ cd aloo
  $ yarn install
```

## Development

```bash
  $ yarn start

  # Testing
  $ yarn test

  # Build
  $ yarn build
```
